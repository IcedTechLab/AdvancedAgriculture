## 如何构建

### 手动构建

```shell
gradle build
```

在工作完成后，最终的构建结果会出现在`./build/libs/`中。

## 作出贡献

您可以通过以下的方法为模组进行贡献：

- 为模组仓库提 PR
- 为模组编写 Wiki
- 为模组进行一定程度的推广
- 为模组的多语言进行贡献
- 为模组所需求的模型，材质等进行设计
- 为我们资助一些经费
- 成为模组开发者的一员
- 提出合理的issues
